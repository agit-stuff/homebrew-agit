# Homebrew Tap for AGIT

This is the official Homebrew tap for [AGIT](https://github.com/agit-stuff/agit), the AI-native Git wrapper.

## Installation

```bash
brew tap agit-stuff/agit
brew install agit
```

## Updating

```bash
brew update
brew upgrade agit
```

## Uninstalling

```bash
brew uninstall agit
brew untap agit-stuff/agit
```
